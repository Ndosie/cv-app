# CV App

A simple React application to build and review a CV (curriculum vitae).

## Features

- Fill in general information, education, and experience sections
- Review entered information

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm start
```

3. Run tests:

```bash
npm test
```

## Project Structure

- `src/` — React source files
- `src/components/` — `General`, `Education`, `Experience`, `Review` components

## Notes

This is a small demo app. Feel free to extend components, add styling, or integrate persistence.
